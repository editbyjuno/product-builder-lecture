# Blueprint: Lotto Number Generator

## 프로젝트 개요
사용자에게 즐거운 시각적 경험을 제공하는 현대적인 로또 번호 생성기입니다. 프레임워크 없이 순수 HTML, CSS, JavaScript와 Web Components 기술을 사용하여 구축됩니다.

## 주요 기능 및 특징
1.  **번호 생성 로직**: 1~45 사이의 중복되지 않는 6개 숫자를 무작위로 추출합니다.
2.  **애니메이션 효과**: 번호가 생성될 때 공이 순차적으로 나타나는 시각적 효과를 제공합니다.
3.  **반응형 디자인**: 모든 화면 크기에서 최적화된 레이아웃을 제공합니다.
4.  **현대적인 UI**:
    *   OKLCH 컬러 공간을 활용한 선명하고 풍부한 색감.
    *   다층 그림자(Multi-layered shadows)를 통한 깊이감 구현.
    *   부드러운 노이즈 텍스처를 배경에 적용하여 고급스러운 느낌 강조.

## 구현 단계
### 1단계: 기본 구조 설정
- `index.html`에 `<lotto-generator>` 컴포넌트 추가.
- `style.css`에 글로벌 스타일 및 배경 텍스처 적용.

### 2단계: Web Component 구현 (`main.js`)
- `LottoGenerator` 클래스 정의.
- Shadow DOM을 사용하여 스타일과 마크업 캡슐화.
- 숫자 생성 알고리즘 및 애니메이션 로직 구현.

### 3단계: 스타일링 (Baseline CSS)
- CSS Variables를 이용한 테마 관리.
- `:has()` 선택자 및 컨테이너 쿼리 활용.
- 버튼 및 공 요소에 Glow 및 Hover 효과 적용.
