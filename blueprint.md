# Blueprint: Lotto Number Generator

## 프로젝트 개요
사용자에게 즐거운 시각적 경험을 제공하는 현대적인 로또 번호 생성기입니다. 프레임워크 없이 순수 HTML, CSS, JavaScript와 Web Components 기술을 사용하여 구축됩니다.

## 주요 기능 및 특징
1.  **번호 생성 로직**: 1~45 사이의 중복되지 않는 6개 숫자를 무작위로 추출합니다.
2.  **애니메이션 효과**: 번호가 생성될 때 공이 순차적으로 나타나는 시각적 효과를 제공합니다.
3.  **테마 시스템 (다크/라이트 모드)**: 사용자의 선호에 따라 테마를 전환할 수 있으며, 선택한 테마는 브라우저에 저장됩니다.
4.  **제휴 문의 폼 (Formspree)**: 사용자가 제휴 문의를 남길 수 있는 폼이며, Formspree를 통해 이메일로 전송됩니다.
5.  **댓글 시스템 (Disqus)**: 사용자 간의 소통을 위한 실시간 댓글 기능을 제공합니다. (외부 Disqus 서비스 연동)
6.  **Google Analytics (GA4)**: 사용자 행동 분석을 위해 Google Analytics 태그를 연동하였습니다.
7.  **반응형 디자인**: 모든 화면 크기에서 최적화된 레이아웃을 제공합니다.
8.  **현대적인 UI**:
    *   OKLCH 컬러 공간을 활용한 선명하고 풍부한 색감.
    *   다층 그림자(Multi-layered shadows)를 통한 깊이감 구현.
    *   부드러운 노이즈 텍스처를 배경에 적용하여 고급스러운 느낌 강조.
